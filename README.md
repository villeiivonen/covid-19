## COVID-19 Demo project



## Built With
* react.js
* TypeScript
* Styled Components
* React Select
* rc-slider

## Prerequisites
1. yarn
  ```sh
  brew install yarn
  ```

## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/villeiivonen/covid-19.git
   ```
3. Install YARN packages
   ```sh
   yarn
   ```

## Usage
1. Run at project root
   ```sh
   yarn dev
   ```
